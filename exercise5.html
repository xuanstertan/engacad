Exercise 5 :

1) Create a new sketch and select the XY-plane as the sketch plane


2) Firstly, I use a two point rectangle with dimensions "40mm" by"130mm


3) Next, I go to the "Create" tab and selected arc > 3point-arc to draw


4) Afterwards, I go the "Create" tab once again to select slot > center to center slot with distance "40mm" and diameter "20mm"

5) Once done, click on the "Extrude" command and change the extrude distance to "10mm"

6) Create a second sketch on the top face of the object


7) Use a "line" command and draw the sketch with dimensions "10mm" by "50mm"

8) Once done, click on the "Extrude" command and change the extrude distance to "30mm"

9) Create a third sketch on the front face of the object and using the "circle" command to draw diameter "30mm" and "50mm"

10) Once done, click on the "Extrude" command and change the extrude distance to "-10mm"

11) Create the final sketch on the front face of the object and draw a "30mm" diameter circle in the center

12) Once done, click on the "Extrude" command and change the extrude distance to "-10mm"

13) Afterwards, do the same for the other side and we are DONE modelling Exercise 5
